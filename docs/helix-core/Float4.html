Class: Float4

Float4

Float4 is a class describing 4-dimensional homogeneous points. These can represent points (w == 1), vectors (w == 0), points in homogeneous projective space, or planes (a, b, c = x, y, z), (w = d).

new Float4(x, y, z, w)

Parameters:
Name Type Description

The x-coordinate
y

The y-coordinate

The z-coordinate
w

The w-coordinate
