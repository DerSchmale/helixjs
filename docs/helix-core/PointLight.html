Class: PointLight

PointLight

PointLight represents an omnidirectional light source with a single point as origin. The light strength falls off according to the inverse square rule.

new PointLight()

Properties:
Name Type Description
radius number

The maximum reach of the light. While this is physically incorrect, it's necessary to limit the lights to a given area for performance.
